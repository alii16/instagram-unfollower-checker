Ekspor Data Pengikut & Mengikuti Instagram

Ikuti langkah-langkah berikut untuk mengunduh data Instagram Anda

Catatan Penting

Waktu Proses: Instagram mungkin memerlukan waktu hingga 48 jam untuk menyiapkan file data Anda. Anda akan menerima notifikasi email ketika file tersebut siap untuk diunduh.

File Format: Pastikan untuk memilih format JSON selama proses ekspor.

Step 1 Step 6 Step 12
1

Buka Pengaturan Instagram

Buka Instagram, klik menu profil Anda, lalu pilih "Pengaturan dan Aktivitas".

2

Akses Pusat Akun

Setelah masuk ke menu pengaturan, "Pusat Akun" pilih di bagian atas.

3

Pilih "Informasi dan Izin Anda"

Cari dan klik opsi "Informasi dan Izin Anda".

4

Pilih "Unduh Informasi Anda"

Klik "Unduh Informasi Anda" untuk mengakses opsi ekspor data.

5

Pilih "Unduh atau Transfer Informasi"

Pilih opsi untuk mengunduh atau mentransfer informasi Anda.

6

Pilih Akun Anda

Pilih akun Instagram yang ingin Anda ekspor data pengikut dan mengikuti.

7

Pilih "Beberapa Informasi Anda"

Pilih "Beberapa Informasi Anda" karena kita hanya memerlukan data pengikut dan mengikuti, bukan semua data akun.

8

Pilih "Pengikut dan Mengikuti"

Centang opsi "Pengikut dan Mengikuti" untuk mengekspor data spesifik ini.

9

Pilih "Unduh ke Perangkat"

Pilih opsi untuk mengunduh data langsung ke perangkat Anda.

10

Isi Formulir

Pengaturan Penting

  • Format: Pilih JSON (bukan HTML)
  • Email: Masukkan email aktif untuk notifikasi unduhan
  • Waktu Proses: Hingga 48 jam

Lengkapi formulir sesuai preferensi Anda lalu klik "Buat File".

11

Ekstrak File yang Diunduh

Setelah Anda menerima email notifikasi unduhan, ekstrak file ZIP untuk mengakses data Anda.

Temukan File Anda

Setelah diekstrak, Anda akan menemukan dua file penting: "followers.json" and "following.json". File inilah yang akan Anda unggah ke pemeriksa unfollowers kami.
